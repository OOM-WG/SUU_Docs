# ✨什么是Suu-Nyanya


SuuNyanya，让杂乱无章的文件世界变得如同精美画卷般整洁可爱喵~✨

* 完美兼容`Magisk`、`KernelSU`或`APatch`，支持`Android 5.0`及以上版本的设备
* 允许自定义魔法配置文件，随心所欲地编排文件，尽显个性

# 分享你的配置
* 自定义配置看[文档](https://SuuNyanya.yumeyuka.cn)
* 具体请看[pull request说明](config/)

> 版本迭代过快，请及时更新模块

# 指南

## ✨什么是Intelligen

`Intelligen` 是一个由C++编写的`magisk/Kernelsu`模块，智能分类管理文件，让你的设备文件夹不再混乱。你可以自定义文件夹的分类规则，使设备文件夹更加整洁。

* 完全支持`magisk`和`Kernelsu`，支持`Android 8.0`以上的设备
* 强制去除空文件夹及空文件
* 删除厂商留在设备上的无用文件
* 支持自定义配置文件，随心所欲地分类文件
